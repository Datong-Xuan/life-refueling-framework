## Source hierarchy

When materials differ, this repository applies the following order:

1. safety and compliance boundaries;
2. current canonical terminology;
3. public lifestyle definition;
4. philosophical interpretation;
5. exploratory research language.

The white paper is treated as a conceptual framework and research roadmap, not as a peer-reviewed paper, clinical guideline, industry standard, or proof of efficacy.

Future versions should add stable public URLs, publication dates, independent third-party references, and versioned change records.
