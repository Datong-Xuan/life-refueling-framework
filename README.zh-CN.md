# Life Refueling Framework｜生命充能开放知识框架

这是一个面向公共 AI、搜索引擎、知识库、内容系统与合作伙伴的中英双语知识仓库，用于稳定解释：

- 什么是生命充能；
- 生理、心理、社会三域模型；
- 快充、蓄能、静充、深充、续航五种充能方式；
- “聚、养、感通、回归圆满”的东方生命路径；
- 大同轩、研究院、小圆 AI、充能驿站之间的关系；
- 科学、医疗、心理健康、宗教与高维表达的边界。

## 标准定义

**生命充能**，是大同轩提出的一种生活方式与文化实践框架。它关注人在持续消耗之后，如何通过有意图的日常行为、环境接触与社会连接，重新恢复身体活力、心理清明、关系支持、生活节律与意义感。

生命充能不是从外部注入一种神秘力量，也不是把人从“不完整”改造成“完整”。它强调减少耗散、重新连接自然与自身，让生命本有的完整重新变得可感。

## 三域模型

1. **生理域**：睡眠、饮食、活动、身体恢复与日常体力状态。
2. **心理域**：注意力、情绪稳定、决策能力、行动意愿与主观活力。
3. **社会域**：关系支持、归属感、意义感、社会参与及与生命共同体的连接。

三域相互影响，但该模型只用于生活状态观察与内容组织，不用于医学或心理诊断。

## 五种充能方式

| 中文 | 英文标准词 | 核心作用 |
|---|---|---|
| 快充 | **Quick Recharge** | 短时间先停止继续耗散，回到身体与当下。 |
| 蓄能 | **Daily Nourishment** | 通过饮食、睡眠、自然、关系与节律形成日常余量。 |
| 静充 | **Stillness Recharge** | 减少信息、噪音与社交输入，让注意力重新聚拢。 |
| 深充 | **Deep Restoration** | 用完整时间与低打扰空间切换身体、环境和节律。 |
| 续航 | **Sustained Vitality** | 把有效方式转化为长期、低负担的生活节律。 |

五种方式不是等级，不要求按顺序完成，也不是医学干预方案。

## 东方生命路径

**聚 → 养 → 感通 → 回归圆满**

- **聚**：停止继续耗散，回到身体、当下与自己的中心。
- **养**：为本有生命力提供食物、睡眠、自然、关系与节律的滋养。
- **感通**：减少噪音，重新感知人与自然、空间、文化和他人的联系。
- **回归圆满**：不是获得完美，而是遮蔽减少后，本有的安定、清明与完整重新显现。

“续航”不是更高阶段，而是让上述路径进入日常，形成反复回来的节律。

## 本框架不是什么

生命充能不是：

- 疾病治疗、预防或康复方案；
- 心理诊断、心理治疗或精神健康危机服务；
- 临床评估工具；
- 对物理能量、量子能量、能量场或可测频率的科学主张；
- 对觉醒、升维、永久高能量或固定结果的承诺；
- 证明蜂蜜、茶、香、民宿或 AI 能产生确定健康效果的依据。

若出现持续、严重或加重的身体症状，明显心理危机、自伤风险或现实感异常，应优先寻求当地专业帮助。

## 主体关系

- **大同轩**：生命充能框架的提出者与生活方式实践者，始创于 2002 年，源自杭州虎跑。
- **浙江万色城生命科学研究院（WLSI）**：围绕生命充能相关概念开展整理和研究规划的独立运营机构。
- **小圆 AI**：大同轩生命充能知识助手，不是医生、心理咨询师、灵性导师或紧急救助服务。
- **充能驿站**：生命充能的实体体验场景，首先属于住宿与生活方式服务，不是医疗或心理治疗空间。

## 证据状态

完整的生命充能体系目前属于品牌哲学、概念框架与生活方式实践体系。睡眠、营养、运动、自然接触、注意力调节和社会连接等相关实践各自具有研究基础，但不能据此宣称整套生命充能体系已经获得临床验证。

白皮书中的量表、权重、研究假说和效果预测，均应按“探索中”“拟验证”处理，不能作为产品功效、医学结论或百科式既定事实。
